Responsive web design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. Recent work also considers the viewer proximity as part of the viewing context as an extension for RWD.

The @media CSS at-rule can be used to apply part of a style sheet based on the result of one or more media queries. With it, you specify a media query and a block of CSS to apply to the document if and only if the media query matches the device on which the content is being used.

Syntax:

The @media at-rule may be placed at the top level of your code or nested inside any other

Media types:

Media types describe the general category of a device. Except when using the not or only logical operators, the media type is optional and the all type will be implied.

Accessibility concerns:

To best accommodate people who adjust a site's text size, use ems when you need a length for your media queries. Both em and px are valid units, but em works better if the user changes the browser text size.

Media queries are useful when you want to modify your site or app depending on a device's general type

Security:

media queries provide insights into the capabilities—and by extension, the features and design—of the device the user is working with.
