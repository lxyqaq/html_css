CSS Flexible Box Layout, commonly known as Flexbox, is a CSS 3 web layout model. It is in the W3C's candidate recommendation (CR) stage. The flex layout allows responsive elements within a container to be automatically arranged depending upon screen size (or device).

The CSS property flex specifies how the elastic element can be stretched or shortened to fit the available space in the flex container. This is a shorthand property to set flex-grow, flex-shrink and flex-basis.

For most purposes, authors should set flex to one of the following values: auto, initial, none, or a positive unitless number.

By default flex items don't shrink below their minimum content size. To change this, set the item's min-width or min-height.

  • Main axis, main dimension. The main axis of the elastic container refers to the direction in which the elastic elements are mainly laid out. It extends in the main axis area.
  • Spindle start point (main-Start), spindle end point (main-end). The elastic elements are placed in the container from the starting point of the spindle to the end of the spindle.
  • Main size, main size property. The main axis size of an elastic element refers to its length in the main axis area. The main axis size attribute refers to its corresponding attribute.
  • Cross axis, cross dimension. The lateral axis is perpendicular to the main axis. It extends in the lateral axis area.
  • Cross-Start, cross-end. Floating rows are filled with elements and distributed along the lateral axis in the container from the start point to the end point.
  • Cross size, cross size property. The lateral axis size of an elastic element refers to its length in the lateral axis area. Its cross-axis size attribute refers to its corresponding attribute

The flex-direction property determines the direction of the main axis (that is, the direction in which the items are arranged).

By default, items are arranged on a line (also known as "axis line"). The flex-wrap attribute defines how to wrap if an axis line cannot fit.

The justify-content attribute defines the alignment of the item on the main axis.It may take 5 values, and the specific alignment is related to the direction of the axis. The following assumes that the main axis is from left to right.
  • flex-start (default): left-justified

  • flex-end: align right

  • center: centered

  • space-between: align at both ends, the space between items is equal.

  • Space-around: The space on both sides of each item is equal.
The flex-grow attribute defines the zoom ratio of the item, and the default is 0, that is, if there is remaining space, it will not zoom.
If the flex-grow property of all items is 1, they will equally divide the remaining space (if any). If the flex-grow property of one item is 2 and the other items are all 1, the remaining space occupied by the former will be twice that of other items.
The flex-shrink attribute defines the reduction ratio of the item, and the default is 1, which means that if the space is insufficient, the item will shrink.
If the flex-shrink property of all items is 1, when the space is insufficient, all items will be reduced proportionally. If the flex-shrink property of one item is 0 and the other items are all 1, the former will not shrink when the space is insufficient.
